# MapleQuest RPG

메이플스토리 스타일의 2D 횡스크롤 RPG 브라우저 게임

## 실행 방법

`game.html` 파일을 브라우저에서 열면 바로 플레이 가능합니다.

## 조작법

| 키 | 동작 |
|----|------|
| ← → / A D | 이동 |
| ↑ / Space / W | 점프 |
| Z | 스킬 1 |
| X | 스킬 2 |
| C | 스킬 3 (버프) |

## 직업

### ⚔️ 전사 (Warrior)
- **특징**: 높은 HP, 느린 속도, 근접 특화
- **스킬**:
  - [Z] 파워 스트라이크 - 강력한 일격
  - [X] 슬래시 블래스트 - 주변 범위 공격
  - [C] 레이지 - 공격력 1.5배 버프

### 🗡️ 도적 (Thief)
- **특징**: 빠른 속도, 높은 크리티컬 (25%)
- **스킬**:
  - [Z] 더블 스탭 - 빠른 2연타
  - [X] 어쌔시네이트 - 백스탭 시 2배 데미지
  - [C] 헤이스트 - 이동속도 1.5배 버프

### 🏹 궁수 (Archer)
- **특징**: 원거리 공격, 높은 공격력
- **스킬**:
  - [Z] 더블 샷 - 화살 2발 발사
  - [X] 애로우 레인 - 화면 전체에 화살비
  - [C] 소울 애로우 - 관통 화살 버프

## 게임 시스템

- **HP/MP**: HP가 0이 되면 부활 (경험치 50% 감소), MP는 자동 회복
- **레벨업**: 몬스터 처치 시 경험치 획득, 레벨업 시 HP/MP/공격력 증가
- **크리티컬**: 직업별 크리티컬 확률로 1.5배 데미지
- **버프**: [C] 스킬로 발동, 좌측 상단에 버프 아이콘 표시
