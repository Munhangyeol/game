# MapleQuest RPG

메이플스토리 스타일의 2D 횡스크롤 RPG 브라우저 게임

## 실행 방법

`game.html` 파일을 브라우저에서 열면 바로 플레이 가능합니다.

## 조작법

| 키 | 동작 |
|----|------|
| ← → | 이동 |
| ↑ / Space | 점프 |
| A | 기본공격 |
| Z | 스킬 1 |
| X | 스킬 2 |
| C | 스킬 3 (버프) |

## 직업

### ⚔️ 전사 (Warrior)
- **특징**: 높은 HP, 느린 속도, 근접 특화. 기본공격마다 적을 밀어내기
- **기본공격**: 검 휘두르기 (강타 + 밀치기)
- **스킬**:
  - [Z] 파워 스트라이크 — 강력한 일격 + 강한 밀치기
  - [X] 슬래시 블래스트 — 주변 원형 범위 공격
  - [C] 레이지 — 공격력 1.5배 버프 (빨간 불꽃 이펚트)

### 🗡️ 도적 (Thief)
- **특징**: 빠른 속도, 높은 크리티컬 (25%), 초고속 연속 기본공격
- **기본공격**: 단검 난무 (초고속 2연타)
- **스킬**:
  - [Z] 삼중 스탭 — 빠른 3연타
  - [X] 어쌔시네이트 — 백스탭 보장, 2배 피해
  - [C] 헤이스트 — 이동속도 1.5배 버프 (보조색 속도선 이펚트)

### 🏹 궁수 (Archer)
- **특징**: 원거리 공격, 높은 공격력, 화살 관통 버프
- **기본공격**: 화살 발사 (원거리 투사체)
- **스킬**:
  - [Z] 더블 샷 — 화살 2발 각도 분산 발사
  - [X] 애로우 레인 — 주변 넓은 범위에 화살비
  - [C] 소울 애로우 — 관통 화살 버프 (영혼빛 이펚트)

## 게임 시스템

- **HP/MP**: HP가 0이 되면 부활 (경험치 50% 감소). MP는 자동 회복
- **레벨업**: 몬스터 처치 시 경험치 획득. 레벨업 시 직업별로 HP·MP·공격력 증가
- **크리티컬**: 직업별 크리티컬 확률로 1.5배 피해. 도적 백스탭은 2배 피해
- **버프**: [C] 스킬로 발동. 좌측 상단 버프 아이콘에 남은 시간 표시
- **몬스터 스케일링**: 레벨이 올라가면 몬스터 HP·피해·경험치도 자동 증가

## 몬스터

|--------|-----|--------|-----------|
| 🟢 슬라임 | 30 + Lv×10 | 25 | 항상 |
| 🔴 머쉬룸 | 50 + Lv×15 | 40 | 항상 |
| 🟤 스텀프 | 80 + Lv×20 | 60 | 레벨 3 이상 |

## 예정 기능

- **전직 시스템**: 레벨 10·30·70에서 전직 가능. 각 직업군마다 고유 성장 경로, 전직마다 새로운 스킬 3개 획득
- **추가 몬스터**: 고레벨 구간별 새 몬스터 타입 (특수 공격 패턴 포함)
- **파일 구조 정리**: 단일 파일에서 모듈로 분리하여 유지보수성 향상
