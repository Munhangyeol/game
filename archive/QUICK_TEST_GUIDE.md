
## 수동 빠른 테스트 (2분)

자동화 대신 수동으로 검증하고 싶다면:

### 전사 테스트
1. http://localhost:8000 열기
2. **전사** 카드 클릭
3. **→** 키로 오른쪽 이동
4. **↑** 키로 점프
5. **A** 키로 공격
6. **Z** 키로 파워 스트라이크 스킬 사용
7. 몬스터가 스폰될 때까지 대기
8. 몬스터 처치 후 경험치 바 채워지는지 확인
9. **C** 키로 레이지 버프 활성화
10. 버프 아이콘이 우측 상단에 나타나는지 확인

**예상:** 모든 것이 부드럽게 작동하고 콘솔(F12)에 에러 없음

### 도적 테스트
위와 동일하지만 **도적** 사용 - 더 빠르게 느껴져야 함

### 궁수 테스트
위와 동일하지만 **궁수** 사용 - 화살이 날아가야 함

## 문제 해결

### 서버가 시작되지 않음
```bash
# 포트 8000이 사용 중인지 확인
netstat -an | findstr :8000

# 다른 포트 사용
python -m http.server 8080
```

### 브라우저에 에러 표시
1. F12를 눌러 개발자 도구 열기
2. 콘솔 탭에서 에러 확인
3. 강제 새로고침 시도: Ctrl+Shift+R
4. 캐시 삭제 후 새로고침

### 테스트 실패
1. 서버가 실행 중인지 확인
2. 모든 버그 수정이 적용되었는지 확인 (BUG_FIXES_REPORT.md 참조)
3. 다른 브라우저 시도 (Chrome/Firefox/Edge)
4. 콘솔에서 구체적인 에러 메시지 확인
